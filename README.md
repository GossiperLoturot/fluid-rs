# fluid-rs

![demo](/img/demo.jpg)

A fluid simulation based on "Becker and Teschner, Weakly compressible SPH for free surface flows, SCA2007".
It use CPU for processing, no use GPU.

# Usage

2D fluid simulation
```sh
cargo run --release --bin 2d
```

3D fluid simulation
```sh
cargo run --release --bin 3d
```
